Dúvidas

PERGUNTAS
FREQUENTES

Preciso saber programar para instalar?+
Não precisa programar. Mas é necessário conhecimento básico de tecnologia — saber seguir um guia, criar conta em plataformas, copiar e colar configurações. Os guias e vídeos foram feitos para facilitar ao máximo esse processo.
Qual é o custo mensal para manter o sistema funcionando?+
O BarberFlow em si não tem mensalidade. Mas as ferramentas que ele usa podem ter custos: n8n no Railway (~US$ 5/mês), OpenAI API (pago por uso, muito baixo para barbearias) e WhatsApp Business API (varia por volume de mensagens). O Supabase e o Google Calendar são gratuitos.
Quanto tempo leva para instalar tudo?+
Em média 2 a 3 horas seguindo os guias. Quem já tem experiência com ferramentas como n8n pode terminar em 1 hora. Os vídeos de instalação foram gravados passo a passo para acelerar esse processo.
Funciona com qualquer número de WhatsApp?+
O kit inclui guia para duas opções: Evolution API (mais fácil, funciona com número comum) e WhatsApp Cloud API (oficial, recomendado para volume maior). O guia explica as diferenças e como configurar cada uma.
E se eu travar na instalação?+
O kit inclui um guia de erros comuns com as soluções mais frequentes. Se precisar de ajuda além do FAQ, você pode contratar o serviço de setup feito por nós a partir de R$ 300.
Precisa de barbeiro específico ou funciona para qualquer tipo?+
Funciona para barbearias de bairro, premium e barbeiros autônomos. O catálogo de serviços e os horários são configuráveis — você adapta para a realidade da sua barbearia.
